## How to Upload to Your Web Host

1. **Upload the entire folder structure** to your web hosting service
2. Make sure the folder structure remains intact:
   - Keep index.html in the root directory
   - Keep css/ folder with style.css inside
   - Keep images/ folder with all the photos inside
3. The index.html file should be in the main/root directory of your domain

## Testing Locally

To test the website on your computer before uploading:

1. Open the `index.html` file in any web browser
2. All features should work including:
   - Navigation
   - Photo toggle buttons
   - Smooth scrolling
   - Animated hearts background
